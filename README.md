# CalorieFetcher — Поиск калорийности и БЖУ продуктов

Этот проект позволяет легко находить информацию о пищевой ценности продуктов — калорийность, белки, жиры и углеводы (БЖУ) — как по штрихкоду, так и по названию. Данные берутся из открытой базы [Open Food Facts](https://world.openfoodfacts.org/).

## Структура проекта

В корне проекта находятся два основных файла:

- `off_rest_example.py` — модуль для взаимодействия с API Open Food Facts.
- `main.py` — графическое приложение на PyQt6 для удобного поиска и отображения данных.

> **ВАЖНО**: в файле `main.py` указан импорт:  
> ```python
> from main import get_product_by_barcode, search_products, extract_kcal
> Без этого графическое приложение не запустится
> ```


## Функционал

### `off_rest_example.py` — API-клиент

Содержит три ключевые функции:

- `get_product_by_barcode(barcode, ...)` — получает данные по штрихкоду.
- `search_products(query, ...)` — ищет продукты по названию.
- `extract_kcal(nutriments)` — извлекает калории и БЖУ из данных.

По умолчанию используется русский язык и фильтр по России.

### `main.py` — Графический интерфейс (PyQt6)

Окно с:

- Полем ввода для названия или штрихкода
- Таблицей результатов
- Детальной информацией по выбранному продукту

Если ввести число — будет поиск по штрихкоду. Иначе — по названию.

## Требования

- Python 3.8 или новее
- Библиотеки: `requests`, `PyQt6`
